# /component - Create New Component

Goal: Scaffold a new React component following project conventions.

Steps:
1. Ask for component name and purpose if not provided
2. Create component file in components/ directory
3. Use TypeScript with proper interface definitions
4. Import from shadcn/ui if UI primitives needed
5. Add JSDoc comment describing the component
6. Export as default
7. Create basic unit test in __tests__/ directory
